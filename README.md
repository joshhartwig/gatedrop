# GateDrop

GateDrop is a fantasy SMX site built with Go, Python, and React. Users can sign up and bet virtual money on real races. At the end of the season, the winner receives a prize.

## Architecture

- **Service**  
  The `service` folder contains a Python service that downloads race results and imports them into a database.

- **API**  
  Manages the backend, including database operations and retrieval of race details, racers, and more.

- **FrontEnd**  
  A React frontend that provides the user experience.
